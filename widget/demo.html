Testing this page outside the dashboard

Serve the file over HTTP and register localhost as a widget domain. Opening it directly as a file:// page does not provide an origin for domain matching.

python3 -m http.server 9000

Then open http://localhost:9000/demo.html. You can also add a public widget key to the script tag below when domain matching is not available.
